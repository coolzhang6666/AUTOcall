# 自动电话拨打系统 (AUTOcall)

![Version](https://img.shields.io/badge/version-3.2.0-blue)
![Android](https://img.shields.io/badge/Android-9%2B-green)
![Kotlin](https://img.shields.io/badge/Kotlin-1.9%2B-purple)

一款功能强大的 Android 自动电话拨打系统，支持批量拨号、通话音频注入、录音等功能。

**GitHub**: https://github.com/ZHCOOL520/AUTOcall

---

## ✨ 核心功能

### 📞 批量拨号
- 支持 Excel (.xlsx/.xls) 和 CSV 格式导入
- 智能识别列标题（电话、姓名、户号、余额等）
- **剪贴板导入**：一键识别并提取电话号码
- 点击联系人直接拨打

### 🔊 音频注入
- 通过 `AudioTrack` 直接注入通话通道
- 本机无外放，仅对方可听到
- 挂断时自动停止播放

### 🎙️ 通话录音
- 实时录音功能（需 ROOT 权限效果更佳）
- MP3 格式，文件自动保存
- 录音路径写入通话记录

### ⚙️ 多卡支持
- 支持选择 SIM 卡 1 / SIM 卡 2
- 双卡交替拨打模式
- 默认卡模式

### 📊 数据管理
- **余额排序**：升序/降序切换
- **拨打统计**：记录拨打次数
- **导出功能**：CSV 格式，Windows 兼容
- **数据持久化**：自动保存联系人和记录

## 🚀 快速开始

### 1. 导入联系人
- **文件导入**：点击"导入电话"，选择 Excel/CSV 文件
- **剪贴板导入**：复制号码后点击"导入剪贴板"

### 2. 配置设置
点击右上角 **⚙️ 设置** 图标：
- **音频播放**：开启后选择语音文件
- **通话录音**：开启后自动录音
- **SIM 卡选择**：指定拨打使用的卡槽

### 3. 开始拨打
- 点击"开始拨打"启动自动拨号
- 可随时暂停/继续/停止
- 查看实时进度和状态

## 📁 文件说明

### 录音文件
- **位置**：`/Android/data/com.example.autocall/files/call_records/`
- **格式**：MP3
- **命名**：`call_号码_时间.mp3`

### 音频文件
- **位置**：`/Android/data/com.example.autocall/files/audio/`
- **支持格式**：MP3, WAV, OGG 等

## ⚠️ 注意事项

### 权限要求
- `CALL_PHONE`：拨打电话
- `READ_PHONE_STATE`：监听通话状态
- `RECORD_AUDIO`：通话录音
- `READ_MEDIA_AUDIO`：读取音频文件

### 系统限制
- **ROOT 设备**：录音和音频注入效果最佳
- **非 ROOT 设备**：部分功能可能受限
- **厂商 ROM**：小米、华为等可能需要额外权限

### 使用建议
- 建议在安静环境测试录音效果
- 使用两台手机互相拨打进行完整测试
- 定期导出通话记录备份数据

## 🐛 常见问题

**Q: 录音无法启动？**  
A: 检查是否授予录音权限，建议使用 ROOT 设备。

**Q: 对方听不到音频？**  
A: 确认音频文件存在，检查音频播放开关是否开启。

**Q: 如何导出联系人？**  
A: 点击底部"导出联系人列表"按钮，选择保存位置。

**Q: 支持哪些号码格式？**  
A: 支持手机号、固话、国际号码（+86/+852 等）、短号码。
